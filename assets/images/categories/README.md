# Category Images

Please add the following image files to this directory:

1. **food.jpg** - Image for the Food category (cooked dishes, pizza, pasta, etc.)
2. **groceries.jpg** - Image for the Groceries category (grocery items in a paper bag)
3. **vegetables_fruits.jpg** - Image for the Vegetables & Fruits category (fresh produce in a basket)
4. **clothing.jpg** - Image for the Clothing & Apparel category (clothing items, jeans, accessories)

The images should be:
- High quality and visually appealing
- Square or landscape orientation (will be cropped to fit)
- Recommended size: 400x400px or larger
- Format: JPG or PNG

The app will automatically fall back to emoji icons if an image file is not found.
